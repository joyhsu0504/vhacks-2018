## Usage

### Bot
1. Go to [DialogFlow](https://dialogflow.com/) and get a key!
2. Clone the repo
3. `cd leif`
4. Place your key in config/api.js
5. `npm install`
6. `npm start`
7. Open Browser , go to "http://localhost:3000"

  Edit the Chat Flow from public/javascripts/convo.js
